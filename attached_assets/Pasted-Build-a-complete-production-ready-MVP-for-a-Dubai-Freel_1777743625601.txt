Build a complete production-ready MVP for a Dubai Freelance Marketplace.

Project objective:
Create a full-stack freelance marketplace for UAE/GCC clients and freelancers, including web frontend, backend API, database, admin panel, authentication, jobs, proposals, chat, payments, ratings, bilingual English/Arabic support, and deployment-ready structure.

Reference scope:
The platform must follow the uploaded proposal requirements for the Dubai Freelance Marketplace MVP, including Next.js web app, Node.js backend, database, JWT authentication, Stripe/PayTabs payment flow, admin panel, profiles, job posting, proposals, messaging, ratings, legal pages, and scalable architecture. :contentReference[oaicite:0]{index=0}

- Frontend: Next.js 14 or latest stable, React, TypeScript, Tailwind CSS
- Backend: Node.js, Express.js, TypeScript
- Database: PostgreSQL with Prisma ORM
- Authentication: JWT with role-based access control
- Realtime: Socket.io for chat
- Payments: Stripe integration first, structure PayTabs as future gateway
