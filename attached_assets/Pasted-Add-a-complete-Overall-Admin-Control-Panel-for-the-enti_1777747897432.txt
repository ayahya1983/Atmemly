Add a complete Overall Admin Control Panel for the entire Khidma marketplace.

Do not rebuild the project.
Keep the existing Khidma backend, Drizzle/Express, and current frontend architecture.

The admin panel must control the whole platform, not only payments.

- RTL by default
- Match the Atmemli-style visual direction
- Clean dashboard cards
- Data tables with filters, search, pagination
- Status badges
- Bulk actions where useful
- Confirmation modals for risky actions

Security requirements:
- Only admin roles can access admin routes
- Apply permission checks to every admin action
- Use Zod validation
- Use Drizzle migrations only
- Do not expose payment secrets
- Add rate limits to sensitive admin actions
- Add audit logs to every mutation
- Prevent accidental deletion by using soft delete where possible

Important:

Use existing models where possible.

The admin panel should allow the platform owner to configure, control, moderate, monitor, and operate the entire Khidma marketplace from one centralized dashboard.
